## Problem Statement

Hospital discharge summaries are often difficult for patients and caregivers to understand. Missing important instructions can lead to medication mistakes, delayed treatment, and avoidable hospital readmissions.

DischargeGuard AI simplifies discharge summaries into easy-to-understand sections including Summary, Warning Signs, Medications, and Caregiver Checklist.

# DischargeGuard AI

AI-Powered Post-Discharge Care Assistant — helps caregivers understand hospital discharge summaries, medications, warning signs, and follow-up instructions.

**Theme:** Crisis Management, HealthTech & Emergency Response

Live Demo Website: **[dischargeguard-ai.vercel.app](https://dischargeguard-ai.vercel.app)**

---

## Features

- Paste or upload discharge summaries (PDF, JPG, PNG)
- Automatic PDF text extraction with OCR fallback for scanned documents
- AI-powered analysis via Hugging Face
- Structured output: Summary, Warning Signs, Medications, Caregiver Checklist

---

## Setup
1. Clone the repository
2. Run:
   npm install
3. Add your Hugging Face token in .env
4. Run:
   npm start
